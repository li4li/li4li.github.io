## 准备工作

* 资源站点 https://www.pinterest.com/
* 关键词 `pixel art` `pixel art 8bit`
* 工具 VSCODE https://code.visualstudio.com/
* WIN10 调出 emoji 快捷键  `WIN+;`
* 注意不要超过16列, 就是宽度不要超过16个emoji, 长度没验证应该无妨
